Estructura de Datos Jerárquicas

Es un modelo de datos en el que los datos se organizan en una estructura similar a un árbol.

Estructura de datos Jerárquicas

Son base de datos que almacenan la información en una estructura jerarquizada, concretamente los datos son organizados de forma parecida a un árbol visto del revés.

Árbol

Es una colección de nodos (a partir de un nodo raíz), donde cada nodo es una estructura de datos con valor, junto con una lista de referencias a los nodos (los hijos).

Árbol binario

Un Árbol binario es una estructura de datos en la cual cada nodo siempre tiene un hijo izquierdo y un hijo derecho. No pueden tener más de dos hijos(nodos).

Tipos de Arboles Binarios

- Árbol Binario Lleno.

- Un Árbol Binario No Lleno.

- Un Árbol Binario Percto y No Prefecto

Árbol De Búsqueda Binario

Es una extensión de árbol binario con varias restricciones opcionales. El valor secundario izquierdo de un nodo debe ser menor o igual que el valor primario y el valor secundario derecho siempre debe ser mayor o igual que el valor primario.

Aplicaciones de los árboles binarios

Árbol Avl

Es un árbol de búsqueda binario que se auto equilibra. En nombre de los inventores Adelson Velshi y Landis, se le da el nombre AVL. Este fue el primer árbol que se equilibró dinámicamente.

Recorrido y Ordenación De Los Nodos

Proceso de visitar de una manera sistematica, exactamente una vez, cada nodo en una estructura de datos de árbol (examinando y/o actulizando los datos en los nodos). Tales recorridos están clasifícados por el orden en el cual son visitados los nodos.

In Order/Entre Orden

Izquierda, Visita y Derecha.

Pre Orden

Visita, Izquierda y Derecha.

Post Order

Izquierda, Derecha y Visita.

El TDA Árbol

Es un modelo que define valores y las operaciones que se pueden realizar sobre ellos. Y se denomina abstracto ya que la intención es que quien lo utiliza, no necesita conocer los detalles de la representación interna o bien el cómo están implementadas las operaciones.

Estructura de Datos Abstracto

En informática, un tipo de datos abstractos (ADT) o tipo de datos abstractos (ADT) es un modelo matemático que consiste en una colección de operaciones definidas en un conjunto de datos en el modelo.

Estructura de Datos Lineales

Son aquellos elementos que ocupan posiciones consecutivas en la estructura, teniendo cada elemento un sucesor y un antecesor.

Estructura de Datos

Las estructuras de datos se basan generalmente en la capacidad de un ordenador para recuperar y almacenar datos en cualquier lugar de su memoria.
